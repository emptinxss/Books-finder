How it works

This applicaton takes data from openlibrary API.
The maximum result you can get is 12 books for each search.

The search bar is set to discover books from their topic.

It can also find some names, titles or authors (not reliable).

The best way, is to use subject!
These are some examples:

love, fantasy, history, horror, kids, adult, warrior, erotic, funny, happy and more!
